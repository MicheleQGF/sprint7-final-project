## 🧠 Objetivo del análisis

- Identificar problemas de calidad de datos
- Analizar comportamientos, distribuciones y outliers
- Visualizaciones de histogramas, boxplots, heatmaps para el análisi
- Generar insights sobre el comportamiento de uso de los clientes y cómo varía entre diferentes grupos de usuarios para el equipo de ConnectaTel.
